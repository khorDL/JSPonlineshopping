# Online Shopping Project (JSP + JDBC + MySQL)
A beginner-friendly JSP-based web application simulating an online shopping platform. This project covers user sign-up, database connectivity using JDBC, and storing user info in a MySQL database.

# Tech Stack
Frontend: HTML, CSS, JSP

Backend: Java (JSP/Servlets), JDBC

Database: MySQL

IDE: Eclipse

Server: Apache Tomcat

Build Tool: None (manual JSP project)

# Prerequisites
JDK 8+

Apache Tomcat 9

MySQL Server

Eclipse IDE for Enterprise Java

JDBC driver: mysql-connector-java-8.0.xx.jar
